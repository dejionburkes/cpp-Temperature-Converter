# Temperature Converter (C++)

A simple command-line program that converts temperatures between Celsius and Fahrenheit.

## What I Learned
- Taking user input in C++
- Working with functions
- Basic arithmetic operations
- If/else control flow

## How to Run
```bash
g++ temperature_converter.cpp -o temp_converter
./temp_converter
```

## Features
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Input validation
- Clean user interface
